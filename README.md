## Observação

Implementei uma ferramenta para a gestão de cardápios do restaurante _Chapa Quente_.
O principal objetivo do projeto é gerar cardápios de forma dinâmica, considerando possíveis restrições alimentares dos clientes e a disponibilidade dos ingredientes em estoque, otimizando a gestão que antes era feita apenas em arquivos CSV.

<br />
<details>
  <summary><strong>:memo: Habilidades trabalhadas </strong></summary>
<br />

- Praticar o conceito de `Hashmaps` através das estruturas `Dict` e `Set`;
- Aplicar conhecimentos de Orientação a Objetos (Classes, Métodos, Instâncias);
- Implementar Testes de Software;
- Lógica de algoritmo para gestão de estoque e receitas;
- Manipulação de arquivos e dados.

</details>
